# 06 — Surveys

**VERA: Volunteer Emergency Response Alliance**

---

## 6.1 Purpose

An online survey was designed to collect quantitative data on how people in Bangladesh experience emergency assistance, their willingness to use VERA, and their priority feature preferences. Results complement qualitative interview findings.

---

## 6.2 Survey Design

| Parameter | Detail |
|-----------|--------|
| **Title** | Emergency Assistance & Volunteer Platform Survey |
| **Target audience** | Citizens, volunteers, donors, students (18+) |
| **Format** | Google Forms / online questionnaire |
| **Question types** | Multiple choice, Likert scale (1–5), short text |
| **Estimated time** | 5–7 minutes |
| **Distribution** | University groups, social media, volunteer networks |

---

## 6.3 Survey Questionnaire

### Part 1 — Demographics
1. Age group: Under 18 / 18–25 / 26–35 / 36–50 / 50+
2. Location: Urban / Semi-urban / Rural
3. Occupation: Student / Employed / Self-employed / Other
4. Have you used social media for emergency help? Yes / No

### Part 2 — Emergency Experience (Likert 1–5)
5. I can find emergency help quickly when needed
6. Blood donors are easy to locate during medical emergencies
7. Disaster relief in my area is well coordinated
8. I trust people who offer help through social media posts
9. I know where my donations go after giving during a crisis

### Part 3 — Platform Willingness
10. Would you register on a centralized emergency platform? Yes / No / Maybe
11. Would you register as a blood donor with automated alerts? Yes / No
12. Would you volunteer if identity verification is required? Yes / No
13. Which features matter most? (Select up to 3)
    - [ ] Emergency requests
    - [ ] Blood donor matching
    - [ ] Volunteer opportunities
    - [ ] Donation tracking
    - [ ] Shelter information
    - [ ] Disaster coverage map
    - [ ] Fundraising campaigns
    - [ ] Certificates for volunteers

### Part 4 — Open Feedback
14. What is the biggest problem with current emergency help channels?
15. What would make you trust an online emergency platform?

## 6.4 Survey Results (Pilot — n = 48)

### Demographics

| Category | Distribution |
|----------|-------------|
| Age 18–25 | 62% |
| Age 26–35 | 25% |
| Urban | 71% |
| Semi-urban / Rural | 29% |
| Students | 54% |
| Used social media for emergencies | 83% |

### Likert Scale Averages (1 = Strongly Disagree, 5 = Strongly Agree)

| Statement | Mean | Interpretation |
|-----------|------|----------------|
| I can find emergency help quickly | 2.1 | **Strong disagreement** — slow response confirmed |
| Blood donors are easy to locate | 1.9 | **Strong disagreement** — blood matching is a top pain point |
| Disaster relief is well coordinated | 2.3 | Disagreement — coordination gaps confirmed |
| I trust social media helpers | 2.5 | Low trust |
| I know where donations go | 2.0 | **Strong disagreement** — transparency needed |

### Platform Willingness

| Question | Yes | Maybe | No |
|----------|-----|-------|-----|
| Register on centralized platform | 79% | 15% | 6% |
| Register as blood donor with alerts | 71% | 18% | 11% |
| Volunteer with ID verification | 65% | 22% | 13% |

### Top Feature Priorities (multi-select)

| Feature | % selecting |
|---------|-------------|
| Blood donor matching | 81% |
| Emergency requests | 77% |
| Donation tracking | 58% |
| Volunteer opportunities | 54% |
| Disaster coverage map | 50% |
| Shelter information | 46% |
| Fundraising campaigns | 42% |
| Certificates | 38% |

### Open Response Themes

| Theme | % of responses mentioning |
|-------|--------------------------|
| Slow response / too much time wasted | 67% |
| Fake or unverified requests | 42% |
| No single platform / too many groups | 58% |
| Donation misuse concerns | 38% |
| Need location-based matching | 50% |

---

## 6.5 Key Survey Conclusions

1. **High willingness to adopt** — 79% would register on a platform like VERA
2. **Blood matching is the #1 priority** — aligns with interview findings
3. **Current channels fail on speed and trust** — all Likert averages below 2.5
4. **Verification is acceptable** — 65% willing to volunteer with ID checks
5. **Transparency is critical** — donation tracking among top feature requests
6. **Urban skew** — future surveys should target rural respondents

---

## 6.6 Survey-to-Requirements Mapping

| Survey finding | VERA requirement |
|----------------|------------------|
| 81% want blood matching | Blood request management, donor search, notifications |
| 77% want emergency requests | Emergency request submission and verification |
| 58% want donation tracking | Donation management and campaign tracking |
| 54% want volunteer opportunities | Volunteer opportunity management |
| 50% want coverage map | Disaster coverage monitoring |
| Low trust in social media | Volunteer verification, request verification |
| 67% cite slow response | Notification system, location-based search |

---

## 6.7 Survey Limitations

| Limitation | Impact |
|------------|--------|
| Sample size n = 48 | Pilot only; not statistically representative nationally |
| Online distribution bias | Over-represents urban, tech-savvy users |
| Self-selection bias | Respondents may already care about emergency issues |
| No longitudinal data | Single point in time |

**Recommendation:** Expand survey to n ≥ 200 with rural outreach before production deployment.
